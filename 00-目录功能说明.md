# 目录功能说明

这个文件夹是共享 AI skills 总库。常用目录和根目录入口含义如下。

## 根目录运行入口

| 文件 | 功能 |
| --- | --- |
| `运行-一键质量闸门.ps1` | 一次性运行本地质量检查。 |
| `运行-接入全部AI平台skills.ps1` | 把标准 skills 接入 Claude Code、Codex 和通用 agents。 |
| `运行-结构审计.ps1` | 检查 skill 目录结构。 |
| `运行-触发词审计.ps1` | 检查触发词和 description 质量。 |
| `运行-导出通用Agents技能清单.ps1` | 生成给通用 agent 读取的 manifest。 |
| `运行-生成真实平台验证包.ps1` | 生成真实 app 测试用 prompt pack 和 CSV。 |
| `运行-汇总真实平台验证结果.ps1` | 汇总真实 app 测试结果。 |
| `运行-严格检查真实平台100%.ps1` | 验证真实多平台是否全部通过。 |

## 目录功能

| 目录 | 功能 |
| --- | --- |
| `00_新skills待整理_INBOX/` | 新 skill 收件夹。用户把新内容先放这里，再让 AI 整理。 |
| `library/skills/` | 标准 skill 库。AI app 和 agents 主要读取这里，每个 skill 必须有 `SKILL.md`。 |
| `library/research-kits/` | 研究资料包、半成品、待整理内容。还不能直接当标准 skill 用。 |
| `library/source-projects/` | 源码项目、第三方工具项目。 |
| `library/references/` | 通用参考资料。 |
| `docs/` | 使用说明、接入指南、技能索引、触发测试报告、真实平台验证结果。 |
| `tools/` | 接入、审计、创建、打包等底层 PowerShell 或 Python 工具脚本。 |
| `archive/` | 原始压缩包、数据库导出、旧版本、备份文件和已处理的新 skill 输入。一般不要删除。 |

## 最常用命令

```powershell
.\运行-一键质量闸门.ps1
.\运行-接入全部AI平台skills.ps1
```
